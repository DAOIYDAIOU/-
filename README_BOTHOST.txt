TRENDSETTER MARKET — ГОТОВО ДЛЯ BOTHOST

1. Загрузи эти файлы в GitHub без вложенной папки.
2. В Bothost подключи репозиторий.
3. Ветка: main
4. Команда запуска: npm start
5. Переменные окружения в Bothost:
   BOT_TOKEN=...
   WEB_APP_URL=https://твой-домен-на-bothost
   PORT=3000
   ADMIN_CHAT_ID=твой_telegram_id
   ADMIN_IDS=твой_telegram_id,второй_id
   ADMIN_KEY=любой_секретный_ключ

ВАЖНО:
- Больше не используй ngrok.
- WEB_APP_URL должен быть доменом Bothost.
- После деплоя открой бота и напиши /start
- Для локального запуска можно использовать SETUP.bat и RUN_ALL.bat
